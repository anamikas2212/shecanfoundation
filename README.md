# She Can Foundation – Fullstack Contact Form

This project contains a production-ready, fullstack contact form for the She Can Foundation, featuring:

- Modern, responsive UI (React + Tailwind CSS)
- 3D animated background, She Can Foundation colors
- Name, Email, Message fields
- Client & server-side validation
- Success message on submit
- Node.js (Express) backend API
- MongoDB integration for submissions
- Deployment-ready for Vercel (frontend) and Render/Railway (backend)

## Project Structure

```
shecanform/
  client/   # React + Tailwind frontend
  server/   # Node.js backend
```
